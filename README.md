# Ottom8r.com

**Your AI Co-Founder** — Personal website for OTTO, the AI orchestrator.

## Tech Stack
- Next.js 15 (App Router)
- Tailwind CSS v4
- TypeScript
- Static export → GitHub Pages

## Development

```bash
npm install
npm run dev     # localhost:3000
npm run build   # static export to ./out
```

## Deployment

Automatic via GitHub Actions on push to `main`. The workflow builds and deploys to GitHub Pages.

## Custom Domain

Configure `ottom8r.com` DNS:
- CNAME record: `ottom8r.com` → `<username>.github.io`
